# 🎓 الطالب (Student)

تخيل معايا إنك طالب في الجامعة، وبدل ما تروح مكتب شؤون الطلبة عشان كل حاجة، هيكون عندك موقع وتطبيق يعمل كل حاجة بضغطة زر. يلا نشوف إيه اللي هتقدر تعمله:

## 📝 التسجيل في المواد (كل ترم)

### الخطوات:

#### 1️⃣ شوف المواد المتاحة ليك

لما تدخل على صفحة التسجيل، النظام **ذكي** وهيديك بس المواد اللي ينفع تاخدها:

```
📚 المواد المتاحة للتسجيل:

✅ CS301 - هياكل البيانات (3 ساعات)
   المتطلبات: ✅ CS201 (خلصتها بـ B+)
              ✅ CS101 (خلصتها بـ A)
   الأقسام المتاحة:

   📌 القسم 1:
      👨‍🏫 الدكتور: د. أحمد محمد
      📅 الأحد والثلاثاء 10:00 ص - 12:00 م
      👥 المقاعد: 12 متبقي من 40
      [اختر هذا القسم]

   📌 القسم 2:
      👨‍🏫 الدكتور: د. سارة علي
      📅 الاثنين والأربعاء 2:00 م - 4:00 م
      👥 المقاعد: ممتلئ (قائمة انتظار)
      [انضم لقائمة الانتظار]

❌ CS401 - هندسة البرمجيات (3 ساعات)
   المتطلبات: ❌ CS301 (لازم تخلصها الأول)
              ✅ CS302
   [لا يمكنك التسجيل الآن]

⚠️ MATH301 - الإحصاء (3 ساعات)
   المتطلبات: ⚠️ MATH201 (راسب - لازم تعيدها)
```

#### 2️⃣ اختار المواد وشوف جدولك

لما تختار مادة، الجدول هيتبني قدامك مباشرة:

```
📅 جدولك المتوقع:

الأحد:
├─ 10:00 - 12:00 │ CS301 - هياكل البيانات │ قاعة A201
└─ 2:00 - 4:00   │ MATH202 - تفاضل وتكامل  │ قاعة B105

الاثنين:
├─ 9:00 - 11:00  │ ENG201 - كتابة تقنية    │ قاعة C301
└─ 1:00 - 3:00   │ LAB - معمل برمجة        │ معمل 3

⚡ تحذير: في تعارض!
   مادة CS305 (الثلاثاء 10:00) تتعارض مع CS301
```

#### 3️⃣ النظام يفحص كل حاجة

قبل ما يسجلك، النظام هيتأكد:

-   ✅ ماخدتش مواد متعارضة في الجدول
-   ✅ خلصت المواد المطلوبة (Prerequisites)
-   ✅ مش زايد عن الحد الأقصى للساعات (18 ساعة مثلاً)
-   ✅ في مكان فاضي في المادة
-   ✅ المادة فعلاً موجودة في منهجك

**لو كل حاجة تمام:**

```
✅ تم التسجيل بنجاح!
   سجلت 5 مواد (15 ساعة)
   📧 هنبعتلك إيميل تأكيد
```

#### 4️⃣ لو غيرت رأيك (فترة الحذف والإضافة)

عندك أسبوعين من بداية الترم تقدر:

-   🔄 تحذف مادة
-   ➕ تضيف مادة جديدة

```
لو حذفت مادة:
├─ قبل أسبوع 1: حذف بدون عقوبة
├─ أسبوع 2-4: حذف مع "W" (منسحب)
└─ بعد أسبوع 4: لازم موافقة المشرف
```

## 📊 متابعة الدرجات

### أثناء الترم:

كل دكتور بيرفع الدرجات على مراحل:

```
📝 CS301 - هياكل البيانات:

├─ Quiz 1: 8/10 ✅
├─ Quiz 2: 9/10 ✅
├─ Midterm: 35/40 ✅
├─ Assignment 1: 18/20 ✅
├─ Assignment 2: قريباً...
├─ Lab: قريباً...
└─ Final: بعد الامتحانات

💯 إجمالي حتى الآن: 70/80 (87.5%)
📈 الدرجة المتوقعة: A
```

### لما الدكتور ينشر الدرجة النهائية:

```
🔔 إشعار جديد!
   د. أحمد نشر درجات مادة CS301

📊 درجتك النهائية:
   ├─ الكويزات: 17/20
   ├─ Midterm: 35/40
   ├─ Final: 75/80
   ├─ المشروع: 38/40
   ├─ الحضور: 10/10
   └─ Bonus: +2 (للمشاركة الممتازة!)

   📌 الإجمالي: 177/190 (93%)
   🎯 التقدير: A
   ⭐ GPA Points: 4.0
```

### لو مش مقتنع بالدرجة؟

```
📝 تقديم تظلم على الدرجة:

1. اختار المادة
2. اختار المكون (Midterm مثلاً)
3. اكتب سبب التظلم
4. ارفع صورة من ورقة الإجابة (لو عندك)
5. اضغط "إرسال التظلم"

📬 الخطوات:
1. تقدم الطلب
2. الدكتور يراجع
3. يوافق أو يرفض
4. هتوصلك رسالة بالقرار
```

## 🎓 اختيار التخصص

في سنة معينة (غالباً تانية أو تالتة)، هتختار تخصصك:

### لما يجي الوقت:

```
🔔 أصبح بإمكانك اختيار تخصصك!

📚 التخصصات المتاحة:

✅ 💻 علوم الحاسب
   ├─ الحد الأدنى للمعدل: 3.0
   │  (معدلك: 3.45 ✅)
   ├─ المتطلبات: CS101, CS201, MATH101
   │  (كلها مكتملة ✅)
   ├─ المقاعد المتاحة: 15 من 50
   └─ [اختر كخيار أول] ⭐

⚠️ 🤖 الذكاء الاصطناعي
   ├─ الحد الأدنى للمعدل: 3.5
   │  (معدلك: 3.45 ❌ ينقصك 0.05)
   ├─ المتطلبات: CS201, CS301, MATH301
   │  (MATH301 غير مكتملة ❌)
   └─ المقاعد: 8 من 30

   💡 نصيحة: احصل على A في الترم الحالي
      عشان يرتفع معدلك ل 3.52!

✅ 🌐 الشبكات والأمن السيبراني
   ├─ الحد الأدنى للمعدل: 2.8 ✅
   ├─ المقاعد: 20 من 40
   └─ [اختر كخيار ثاني]
```

### خطوات التقديم:

```
1️⃣ رتب اختياراتك:
   ├─ الخيار الأول: علوم الحاسب ⭐
   ├─ الخيار الثاني: الشبكات
   └─ الخيار الثالث: نظم المعلومات

2️⃣ اكتب ليه عايز التخصص ده (اختياري)

3️⃣ ارفع مستندات (لو مطلوبة)

4️⃣ اضغط "إرسال الطلب"

📬 هيروح للإدارة
   └─ يراجعوا طلبك
       └─ يوافقوا أو يرفضوا
           └─ هتوصلك رسالة بالقرار
```

### بعد الموافقة:

```
🎉 مبروك!
   تم قبولك في تخصص علوم الحاسب

📚 المواد اللي لازم تخلصها:
   └─ 42 ساعة متبقية من التخصص
```

## ❌ ما لا تقدر تعمله (محظور)

### 🚫 لا يمكنك:

```
❌ نشر الدرجات النهائية
   └─ الدكتور فقط يقدر ينشر الدرجات

❌ تعديل درجات Midterm/Final
   └─ حتى لو رفعتها، الدكتور لازم يوافق

❌ منح Bonus للطلاب
   └─ لازم موافقة الدكتور

❌ قبول/رفض أعذار الغياب
   └─ الدكتور يقرر

❌ تعديل إعدادات المادة
   └─ توزيع الدرجات، السياسات، etc.

❌ حذف محتوى رفعه الدكتور
   └─ تقدر بس تحذف اللي رفعته أنت

❌ الوصول لدرجات طلاب سكاشن تانية
   └─ سكشنك فقط

❌ إرسال إعلانات رسمية
   └─ تقدر بس ترد على الاستفسارات
```
